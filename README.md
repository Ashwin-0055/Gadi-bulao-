## Features

- Book rides with Bike, Auto, or Cab
- Real-time driver tracking on map
- OTP verification for ride security
- Driver earnings dashboard
- Admin panel to manage rides & users

---

## Tech Stack

- **Mobile App:** React Native, Expo, TypeScript
- **Backend:** Node.js, Express.js, MongoDB
- **Real-time:** Socket.io
- **Maps:** OpenStreetMap
