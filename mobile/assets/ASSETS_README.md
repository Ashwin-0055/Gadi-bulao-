# Mobile App Assets

This directory contains image assets required for the mobile application.

## Required Assets

The following assets need to be added to this directory:

### Map Markers

1. **pin.png** (48x48px)
   - Used for the draggable pickup location pin in customer home screen
   - Should be a pin/marker icon
   - Recommended: Green color (#4CAF50)

2. **car-marker.png** (40x40px)
   - Used for showing driver's location on the map
   - Should be a car icon (top-down view)
   - Recommended: Black or dark color

## How to Add Assets

### Option 1: Use Free Icon Resources

You can download free icons from:
- **Flaticon**: https://www.flaticon.com/
- **Icons8**: https://icons8.com/
- **Freepik**: https://www.freepik.com/

Search for:
- "map pin icon"
- "location marker icon"
- "car top view icon"

### Option 2: Create Custom Icons

Use design tools like:
- Figma (free)
- Canva (free)
- Adobe Illustrator

### Option 3: Use Emoji-based Fallback

If you don't have custom assets, the app will work but may look less polished. You can temporarily use emoji or text-based markers.

## Expo Default Assets

The following assets are auto-generated by Expo:
- **icon.png** - App icon (1024x1024px)
- **splash.png** - Splash screen
- **adaptive-icon.png** - Android adaptive icon
- **favicon.png** - Web favicon

You can customize these later using the Expo Assets documentation:
https://docs.expo.dev/develop/user-interface/splash-screen-and-app-icon/

## Notes

- All PNG files should have transparent backgrounds where appropriate
- Use appropriate resolution for different screen densities
- Consider creating @2x and @3x versions for better quality on high-DPI screens
